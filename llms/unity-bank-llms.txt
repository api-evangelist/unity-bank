# Unity Bank

> Unity Bank Limited (ABN 72 087 650 637, AFSL & Australian Credit Licence 238311, BSB 659 000) is an Australian member-owned mutual bank (ADI). Under the Australian Consumer Data Right (CDR / open banking) it exposes a public, unauthenticated Product Reference Data (PRD) API conforming to the DSB Consumer Data Standards, served on a shared CDR platform. No credentials are required to read product data.

## API

- [Unity Bank CDR Product Reference Data API](https://www.unitybank.com.au/about-us/corporate-information/public-apis/): Public, unauthenticated CDR PRD. Base URL https://ibank.gcmutualbank.com.au/openbanking/cds-au/v1. GET /banking/products and GET /banking/products/{productId}. Requires x-v header (supported 4, 5). Brand code UBL; 74 products live.
